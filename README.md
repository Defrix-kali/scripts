# Kali Linux Automation Scripts

Набор скриптов для автоматизации настройки окружения, смены сетевой личности и обеспечения анонимности.

## Включенные инструменты:
* **fan.sh** — Скрипт для полной анонимизации (смена MAC-адресов всех интерфейсов, маскировка Hostname, отключение IPv6 и перенаправление TCP-трафика через сеть Tor).

## Использование:
```bash
sudo ./fan.sh
\```
